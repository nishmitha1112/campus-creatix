# Campus Creatix

A full-stack web platform that enables students and faculty to share campus posters and announcements in a centralized system.

---

## 🔗 Live Demo
[Campus Creatix Live](https://app1-42aaa.web.app/#/login)
---

## Problem Statement

In many colleges, event posters and announcements are shared via WhatsApp groups or physical notice boards. This leads to:
- Missed updates
- Poor communication
- Lack of engagement

---

## 💡 Solution

Campus Creatix provides a centralized digital platform with role-based dashboards:
- 👨‍🎓 Student Dashboard
- 👨‍🏫 Faculty Dashboard

---

## ✨ Features

### Student Dashboard
- Create & upload posters
- View campus posts
- Like & rate posters

### Faculty Dashboard
- Post official announcements
- Manage content
- Monitor student activity

---

##  Tech Stack

- Frontend: Angular
- Backend: Firebase
- Database: Firestore
- Authentication: Firebase Auth
